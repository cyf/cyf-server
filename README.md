# server

## Installation

```bash
pnpm i
```

## Running the app

```bash
# development
pnpm start

# watch mode
pnpm start:dev

# production mode
pnpm start:prod
```

## Endpoint

### Swagger

GET: `http://127.0.0.1:3001/api`

## Test

```bash
# unit tests
pnpm test

# e2e tests
pnpm test:e2e

# test coverage
pnpm test:cov
```
